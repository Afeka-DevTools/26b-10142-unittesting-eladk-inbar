## 🚀 How to Run the Tests

Follow these instructions to clone the project to your local machine and execute the unit test suite.

### 1. Prerequisites
Ensure you have the following installed on your personal computer:
* [Git](https://git-scm.com/)
* [Java Development Kit (JDK)](https://www.oracle.com/java/technologies/downloads/)
* [Gradle](https://gradle.org/install/) (Optional: You can use the included Gradle Wrapper instead)

### 2. Clone the Repository
Open your terminal or command prompt and run the following commands to clone the repository and navigate into the project directory:

```bash
git clone <repository_url>
cd <repository_directory>
```

### 3. Execute the Tests
To run all the tests written for this project, run the following command. 

If using the **Gradle Wrapper** (Recommended):
* **Windows:**
  ```cmd
  gradlew test
  ```
* **Mac/Linux:**
  ```bash
  ./gradlew test
  ```

If you have Gradle installed globally:
```bash
gradle test
```

### 4. View Test Reports
Once the tests finish running, Gradle will automatically generate a detailed HTML report. You can view the test results and coverage by opening the following file in your web browser:
`build/reports/tests/test/index.html`
